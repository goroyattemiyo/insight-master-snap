# SCREEN_HOME

## 目的
ユーザーが迷わず最初の分析を始められること。

## 画面の役割
- アプリの価値を一目で伝える
- スクショ選択へ最短で導く
- 対応SNSを明示する
- 設定不要の安心感を与える

## 表示要素
- ブランド名: InsightMasterforAndroid
- キャッチコピー
- メイン見出し
- 説明文
- メインCTA「スクショを選択」
- 対応SNSチップ
- 下部ナビゲーション

## UX方針
- 1タップで開始できる
- CTAを最も目立たせる
- 情報は少なく理解は速く
- 片手操作を前提にする

## 注意
- 旧名称 The Insight Editorial は使用しない
